# Services
여러 기능들을 제공하는 사이트를 만드려고 합니다.

**1. Random Media**
- 🎞 랜덤 재생
- 💾 필요하면 다운로드 가능

## Random Media
### 1) Random Video
**서비스 특징**
- Pixabay의 Video API를 이용한 여러 영상 노출
- 카테고리별 20개의 랜덤 영상 노출
- 다양한 해상도 제공(`tiny`, `small`, `medium`, `large`)

**추가 기능 예정**
- 더 구체적인 키워드를 적용하여 노출

### 2) Random Music
**서비스 특징**
- 비공식 Pixabay의 Music API를 이용한 여러 음원 노출
- 최대 128개의 장르의 음악

**추가 기능 예정**
- 태그 데이터 구체화
